# Компьютерное зрение, процессы связанные с видеоанализом.

## Описание проекта
 Компьютерное зрение — это одна из наиболее активно развивающихся областей, в контексте нейросетей, которая находит применение в самых разных сферах: от анализа изображений до обеспечения безопасности.
Работа с базовыми фильтрами и методами компьютерного зрения не требует мощных ресурсов.
Задача, определить где на фотографии обозначена окружность.

###  Создан через Python-код в Google Colab.

Google Colab - облачная платформа для разработки и выполнения Python-кода в браузере.
